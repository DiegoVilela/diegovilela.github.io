Hi, I’m Diego. Welcome to my site.

Stack

Python is my preferred programming language at the moment, but I’m also interested in Rust and Javascript.

I mostly use the Django web framework in my projects and Bootstrap for the front-end. htmx has caught my attention and I’m looking forward to using it in production as it works very smoothly with the Django template system.

I use Docker during development and usually deploy to AWS (EC2 with elastic IP address, ECR to store built images, and RDS to serve Postgres databases), Elastic Beanstalk or Heroku.
